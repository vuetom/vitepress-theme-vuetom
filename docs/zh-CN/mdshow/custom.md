# 自定义语法

## 信息框

```md
::: tip 使用TIPS代替
提示信息
:::

::: info
信息消息
:::

::: warning
警告消息
:::

::: danger
危险消息
:::
```
效果如下：

::: tip 使用TIPS代替
提示内容
:::

::: info
INFO消息
:::

::: warning
WARNING消息 <a>a链接</a>
:::

::: danger
DANGER消息 [md链接]()
:::
