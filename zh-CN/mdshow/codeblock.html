MD语法展示 #

使用三个 ` 符号将代码包裹其中


亮色主题

可以在 language_key(语言标识) 后加入 light

输入内容：

```java light
String language = "Java";
```

输出内容：

String language = "Java";

暗色主题

默认的话就是暗色系

输入内容：

```js
String language = "JS";
```

输出内容：

String language = "JS";
